## 1. Clone -> cd fastbill_be
## 2. Rename file: .exam.env => .env 
## 3. Add Environment variable .env
## 4. Install node_modules
```bash
    npm install
```
## 5. cd src -> run in Terminal: 
```bash
    npx sequelize-cli db:migrate
```
## 6. Run project
```bash
    npm start
```
